THIS FOLDER CONTAINS THE DIFFERENT SOURCE FILES AND HEADERS THAT CORRESPOND TO
THE PLAYER'S ATTRIBUTES WHEN THEY CREATE A CHARACTER. IT NEEDS TO HAVE
THE FOLDER NAME INCLUDED IN THE REFERENCE CALL IN THE HEADER INCLUDE
IN THE CHARSETTER SOURCE FILE AND ANY OTHER ASSORTED FILES THAT
CALL THOSE HEADER FILES. THIS WILL ALSO INCLUDE THE PLAYER HEADER
FILE.
